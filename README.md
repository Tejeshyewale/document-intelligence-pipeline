# 📄 Document Intelligence Pipeline

A production-grade NLP pipeline built with RocketRide that transforms 
any PDF or text document into structured intelligence.

## Features
- 📝 Abstractive Summarization (BART)
- 😊 Sentiment Analysis (RoBERTa)  
- 🏷️ Named Entity Recognition (spaCy)
- 💬 Q&A Chatbot (RoBERTa SQuAD2)
- 🔗 Built on RocketRide pipeline architecture
